What is Logservice ?

flash_on

Parse and send your cloud foundry logs fastly

Logservice is here to parse cloud foundry logs and format them in json and finally forward them in syslog rfc5424 format to a or multiple log storage platform(s) in same time. (e.g.: Elastic stack).

group

User Experience Focused

It lets user define is own parsing rule in grok format but also tags which will be include in logs.

settings

Easy to work with

Logservice implements service broker api for make use this service simply as you will works with others services. In addition, logservice parse itself router, metrics and common app logs like apache without do anything. Operators can also include parsing rules by default for easy usage.
